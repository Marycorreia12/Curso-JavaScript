Variavéis

Existem 3 maneiras de declarar uma variável JavaScript:

  1. Usando var
  2. Usando let
  3. Usando const

Nesta aula vou aprender como usar a variavél const.

Prima f12 para ver o resultado no console do navegador.

Algumas regras do uso da variável const

As variáveis definidas com const não podem ser redeclaradas, nem reatribuídas.

Não podemos criar constantes com palavras reservadas como o próprio let e const.

Constantes precisam ter nomes significativos.

Não podemos começar o nome de uma constantes com um número.

Utilizamos camelCase em nomes grandes.

Const é um case-sentitive.

Não podemos modificar o valor de uma constante.

A variáveis definidas com const tem bloco de escopo, variáveis declaradas dentro de um {} não podem ser acessadas fora do bloco.

Com const tb não é permitido redeclarar uma variável no mesmo bloco, em outro bloco é permitido.

Usar uma variável const antes de ser declarada resultará em ReferenceError.
