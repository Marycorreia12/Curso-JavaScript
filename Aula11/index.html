Funções JavaScript

Uma função JavaScript é um bloco de código projetado para executar uma tarefa específica.

Uma função JavaScript é definida com a palavra - chave function, seguida por um nome , seguida por parênteses().

Os parênteses podem incluir nomes de parâmetros separados por vírgulas.

O código a ser executado, pela função, é colocado entre chaves: {} e será executado quando "algo" invocar (chamar) a função.

Quando o JavaScript atinge uma instrução return, a função para de ser executada.

Variáveis ​​declaradas em uma função JavaScript tornam-se LOCAIS para a função.

Variáveis ​​locais só podem ser acessadas de dentro da função.

// code here can NOT use carName function myFunction() { let carName = "Volvo"; // code here CAN use carName } // code here can NOT use carName
