Variavéis

Existem 3 maneiras de declarar uma variável JavaScript:

  1. Usando var
  2. Usando let
  3. Usando const

Nesta aula vou aprender como usar a variavél var.

Prima f12 para ver o resultado no console do navegador.

Não utilize var utilize let.

Com var é possível redeclarar variáveis em qualquer lugar em um programa.

Com var voçê pode usar a variável antes de ser declarada.

A variáveis definidas com var tem bloco de escopo, variáveis declaradas dentro de um {} podem ser acessadas fora do bloco.
