Alert

O método alert () exibe uma caixa de alerta com uma mensagem especificada e um botão OK.

Uma caixa de alerta é freqüentemente usada se você quiser garantir que as informações cheguem ao usuário

Confirm

O método confirm () exibe uma caixa de diálogo com uma mensagem especificada, junto com um OK e um botão Cancelar.

Uma caixa de confirmação é freqüentemente usada se você quiser que o usuário verifique ou aceite algo.

O método confirm () retorna verdadeiro se o usuário clica em "OK" e falso caso contrário.

Prompt

O método prompt () exibe uma caixa de diálogo que solicita a entrada do visitante.

Uma caixa de prompt é freqüentemente usada se você deseja que o usuário insira um valor antes de inserir uma página.

O método prompt () retorna o valor de entrada se o usuário clicar em "OK". Se o usuário clicar em "cancelar", o método retornará nulo.
