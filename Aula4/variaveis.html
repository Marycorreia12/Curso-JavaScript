Variavéis

Existem 3 maneiras de declarar uma variável JavaScript:

  1. Usando var
  2. Usando let
  3. Usando const

Nesta aula vou aprender como usar a variavél let.

Prima f12 para ver o resultado no console do navegador.

Algumas regras do uso da variável let

Não podemos criar variáveis com palavras reservadas como o próprio let e const.

Variáveis precisam ter nomes significativos.

Não podemos começar o nome de uma variável com um número.

Utilizamos camelCase em nomes grandes.

Let é um case-sentitive.

Não podemos redeclarar variáveis com let.

Não utilize var utilize let.

A variaáveis definidas com let tem bloco de escopo, variáveis declaradas dentro de um {} não podem ser acessadas fora do bloco.

Com let não é permitido redeclarar uma variável no mesmo bloco, em outro bloco é permitido.
